FAQ

Clear answers before checkout.

Is Managerarr tied to one operating system?

No. The main sellable runtime is built to work across macOS, Windows, and Linux.

How does the demo work?

The site no longer exposes a public demo download. Demo access is by request, which is safer than leaving files openly linked on the public website.

How do I know a buyer actually paid?

You issue a signed license token after checkout. Managerarr verifies the signature offline before unlocking full mode.

Is setup too technical for regular buyers?

No. The runtime creates a starter config and quick-start guide automatically, so the setup path stays short and predictable.

Does this release require anything preinstalled?

Yes. The current cross-platform release assumes the buyer has Node.js 20 or newer installed, along with the apps or commands they want Managerarr to launch.

Can a static website completely stop copying?

No. True purchase-only delivery requires private file hosting, checkout-gated fulfillment, and expiring or authenticated download links. The current site is updated to support that flow, but the hard security boundary must live in the delivery backend or storefront.
